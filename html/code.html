OPEN SOURCE SOFTWARE

MESH-SED

MESH-SED is a physically-based sediment transport model for large scale catchments. The model estimates the hillslope and the in-stream transport of sediments and report sediment concentration at every cell of the computational mesh used by the hydrological model. The model estimates sediment concentration for a large number of sediment grain sizes and it is loosely coupled to MESH, one of the Environment and Climate Change Canada hydrological models.

UCLAKE

UCLAKE is a 1D lake model that estimate the thermal structure in lakes. The model discretizes a lake basin domain in different layers that dynamically resize upon changes in lake volume. The model is computationally efficient and is suitable to investigate long-term changes in lake thermal structure.

[1] Morales-Marín L, French J, Burningham H, Evans C and Burden A. Simulating seasonal to multi-decadal variation in lake thermal response to meteorological forcing using the UCLAKE 1-dimensional model code. Limnologica

FVCOM-GUI

FVCOM-GUI is a graphical user interface (GUI) used to pre- and post-processing FVCOM model input and output data. The GUI is programmed in Matlab and offers different options to display FVCOM output data saved in a NetCDF file.

UCL-SWM

UCL-SWM is a wind-wave hindcasting model to propagate surface water waves. The model estimate wave period, height and amplitude across a water body domain. It is suitable to estimate wind-wave characteristics in lakes.
